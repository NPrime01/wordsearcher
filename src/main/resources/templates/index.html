How to Use the UI

  1. Click Here to begin
  2. Type your query into the input box
  3. Select Which mode you would like to query with
  4. Press the Submit Query Button
  5. You can browse through your Query results below or copy them to clipboard using the Copy to clipboard button

How to use the API

  • With the application running, just send a GET request to localhost:8080/api/[Mode]?word=[query]
  • Simply replace [Query] with the word you would like to search for
  • Replace [Mode] with the mode you would like to use,
    • sw for Starts With mode
    • ew for Ends With mode
    • co for contains mode

The response will have the following structure: 

{
    'word': 'your_word',
    'mode': 'search_mode',
    'results': ['list', 'of', 'results']
}
